# WobbbleBlog-v1
Very basic Node + Express blog website with some kinda crude CRUD. non persistent since i never hooked it up to a database

node_modules not included since package.json has the dependencies, and i dont wish to compete with TON 618.

<ol>
  <li>Install dependencies via: </li>
  <code>npm i</code>
  <br>
  <li>Run via: </li>
  <code>node index.js</code>
</ol>
